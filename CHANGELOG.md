# Changelog

Balli follows semantic versioning while it remains pre-1.0: minor versions may
add features and refine APIs, patch versions are reserved for bug fixes.

## 0.6.0

- Added Malli parity phase 6: map/AST syntax, lite syntax, `:seqable`,
  `:every`, `:->`, var registries, generator objects, destructuring, and
  compatibility helper namespaces.
- Added expanded fuzz/stress and user-suite coverage.

## 0.5.0

- Added Tier 5 hardening, fuzz/stress coverage, and adversarial review fixes.

## 0.4.0

- Added Tier 4 Malli ports.

## 0.3.0

- Added Tier 3 schema functionality and quick user confidence tests.
